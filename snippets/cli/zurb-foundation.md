# Clone and build a Foundation starter template

```sh
git clone https://github.com/foundation/foundation-sites-template.git ./suites
```

```sh
cd suites
```

```sh
rm -rf .git
```

```sh
yarn upgrade
```

```sh
yarn start 
```

Versions I used (as of 2022-03-30):
* node  v14.18.1
* npm   v8.5.5
* nvm   v0.37.2
* yarn  v1.22.15
